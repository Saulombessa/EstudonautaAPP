# Estudonauta APP :iphone:
## Curso estudonauta modulo 02
### App para divulgar a staff e serviços do site ESTUDONAUTA

O aplicativo foi elaborado para treinar recursos do programa ANDROID STUDIO, durante o processo de criação, tive contato com os seguintes conceitos:
- Interfaces
- Personalização do ícone do aplicativo
- Acréscimo de imagens estáticas  
- Leyouts lineares 
- Formatação de layouts
- Botões que chamam outras Atividades (telas)
- App com vídeo (auto play)
- Navegação externa
- onClick em imagens
- Rolagens horizontais e verticais
